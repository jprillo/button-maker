How to generate the CSS for your HTML buttons

Use the controls to make the button look the way you want. Then generate the CSS. Change the class name to whatever you want. Adjusting the text size changes all of the styles. That means you only have to change to text size in a media quiery and it changes everything. ENJOY.

Why use a CSS generator for your links and buttons

I made this so I can prototype your html button and anchor style without having to use a graphic design program first. It is also handy when someone gives you an image to code. This can help get things like padding and box-shadow right without having to refresh the browser.
